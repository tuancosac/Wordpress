== Description ==

SKT White is a free responsive WordPress theme which is flexible, simple, adaptable, artistic, clean, creative, fullscreen, modern, portfolio, elegant, stylish, multipurpose, easy, minimalistic, simplistic, sleek, plain, minimal and clear for any type of website you want to create. So corporate, business, photography, wedding, industrial, restaurant or any sort of industry. SKT White has the ability to change the link colors. So make it suit your style you want. Built with page builder. WooCommerce and SEO compatible. Translation ready.
